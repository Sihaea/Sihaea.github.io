About



What is Flow?

Flow is a psychological state of optimal performance characterised by intense focus, seamless execution, and a distorted sense of time. Originating from the work of Mihaly Csikszentmihalyi, flow is a proven enhancer of skill acquisition and creativity.

Project Outline:

This project investigates how flow can be stimulated and applied in language learning through the combined use of adaptive deep learning models and tDCS technology. The goal is to create a system that dynamically responds to user metrics, sustaining flow and enhancing language acquisition.
